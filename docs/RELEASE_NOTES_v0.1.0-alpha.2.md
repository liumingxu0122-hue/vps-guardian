# v0.1.0-alpha.2 release notes draft

## English

This alpha adds first-class English and Simplified Chinese support across the Dashboard and core documentation. It includes browser-language detection, a persistent manual selector, Intl-based formatting, localized operational states, and bilingual UI tests and screenshots.

Installation starts with [the English quick start](en/QUICKSTART.md). The limitations of the Developer Preview remain unchanged; production use is not recommended.

## 简体中文

本 Alpha 为 Dashboard 和核心文档新增完整的 English / 简体中文支持，包括浏览器语言检测、手动选择持久化、基于 Intl 的格式化、本地化运维状态，以及双语 UI 测试和截图。

安装请从[简体中文快速开始](zh-CN/QUICKSTART.md)开始。Developer Preview 的现有限制保持不变，仍不建议用于生产环境。
